# ChainIDE showcase planetnft

This is a project for ChainIDE showcase.

### develop

#### 1. Build & start backend (NFT Metadata Storage)

```
cd backend
yarn install
yarn build
yarn start
```

URL: http://localhost:3001

#### 2. Build & start frontend (NFT Mint Page)

```
cd frontend
yarn install
yarn build
yarn start
```

URL: http://localhost:3000
